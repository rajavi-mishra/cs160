Electric Time Converter: An Android App

Electric Transport Converter is an android application that allows the user to view how much time it would take them to travel a given distance using an electric vehicle. For this, users have to select their desired mode of electric transport, input the distance they want to travel and click convert. This would show the amount of time it will take the users to travel the inputted distance unless it is out of the traveling distance for that particular mode of transport. The users will also know how long it will take them to travel that distance if they pick a different mode of electric transport, allowing them to efficiently compare and choose from multiple options. Users can use this functionality in both landscape and portrait modes.
